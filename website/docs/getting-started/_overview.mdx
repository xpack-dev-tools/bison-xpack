The main target of this project is macOS 10.13, as the older version of
this utility fails during some builds.

[GNU bison](https://www.gnu.org/software/bison/) is a general-purpose parser
generator that converts an annotated context-free grammar into a deterministic
LR or generalized LR (GLR) parser employing LALR parser tables.

**GNU bison** is an open source project hosted on
[Savannah](https://savannah.gnu.org/projects/bison).
